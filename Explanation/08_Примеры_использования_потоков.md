# Слайд 8: Примеры использования потоков

Этот слайд демонстрирует практические примеры использования потоков в Node.js.

Системные вызовы, использующие потоки Libuv:
1. Файловая система (fs.readFile, fs.writeFile)
2. Сетевые операции (http, https)
3. Шифрование (crypto)
4. Работа с базами данных
5. Парсинг данных (JSON, CSV)

Пример использования Worker Threads:
```javascript
const { Worker, isMainThread, parentPort } = require('worker_threads');

if (isMainThread) {
  const worker = new Worker(__filename);
  worker.on('message', (msg) => {
    console.log('Received:', msg);
  });
} else {
  parentPort.postMessage('Hello from worker');
}
```

Когда использовать Worker Threads:
- Вычислительно intensive задачи
- Обработка больших объемов данных
- Математические вычисления
- Работа с изображениями
- Парсинг и анализ данных

Основное различие:
- Системные потоки для I/O операций
- Worker Threads для CPU операций
