# Слайд 19: Архитектура Libuv

Этот слайд детализирует архитектуру Libuv и его компоненты.

Основные компоненты Libuv:

1. Event Loop:
   - Центральный элемент управления
   - Очередь событий и задач
   - Переключение между состояниями
   - Обработка таймеров и I/O

2. Thread Pool:
   - По умолчанию 4 потока
   - Конфигурируемый через UV_THREADPOOL_SIZE
   - Обработка блокирующих операций
   - Очередь задач для потоков

3. I/O Overlays:
   - Абстракция системных вызовов
   - Обработка файловых операций
   - Сетевые операции
   - Системные вызовы ОС

4. Async Handle:
   - Обработка асинхронных событий
   - Пользовательские обработчики
   - Сигналы и другие события

5. Timer Handle:
   - Управление таймерами
   - Планирование задач
   - Обработка задержек

Работа компонентов:
- Event Loop получает задачи от Node.js
- Системные вызовы обрабатываются через I/O Overlays
- Блокирующие операции направляются в Thread Pool
- Все результаты возвращаются через Event Loop обратно в JavaScript
