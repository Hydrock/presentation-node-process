# Слайд 2: Процессы vs Потоки

Этот слайд подробно объясняет различия между процессами и потоками.

Процессы:
- Независимая единица выполнения с собственной памятью и ресурсами
- Высокая стоимость создания и переключения контекста
- Изоляция ресурсов между процессами
- Межпроцессное взаимодействие требует специальных механизмов (IPC)
- Пример: каждый запущенный Node.js процесс как отдельный процесс

Потоки:
- Внутрипроцессовые единицы выполнения, делящие память и ресурсы
- Низкая стоимость создания и переключения контекста
- Общий доступ к ресурсам внутри процесса
- Межпотоковое взаимодействие более простое, но требует синхронизации
- Пример: потоки в Libuv для асинхронных операций

В Node.js:
- JavaScript выполняется в одном потоке (Event Loop)
- Многопоточность реализуется на уровне системы через Libuv
- Это позволяет эффективно обрабатывать I/O операции без блокировки основного потока
