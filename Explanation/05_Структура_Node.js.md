# Слайд 5: Структура Node.js

Этот слайд демонстрирует архитектуру Node.js и как она организует выполнение.

Архитектурные компоненты Node.js:
1. Application - пользовательский код
2. JavaScript - код на JavaScript
3. V8 Engine - движок JavaScript (Google's V8)
4. Node.js API - нативные модули Node.js
5. Libuv - библиотека асинхронного ввода-вывода
6. OS/Kernel - операционная система и ядро

Важные аспекты:
- V8 выполняет JavaScript код в одном потоке
- Node.js API предоставляет доступ к системным вызовам
- Libuv обеспечивает асинхронность на уровне системы
- Все вызовы I/O направляются через Libuv
- Благодаря этому Node.js может эффективно обрабатывать множество одновременных операций

Связь между компонентами:
- JavaScript вызывает Node.js API
- Node.js API использует Libuv для асинхронных операций
- Libuv взаимодействует с ОС для выполнения системных вызовов
- Результаты возвращаются обратно через Event Loop
